# Claude Code Study

이 저장소는 **Claude Code를 효율적으로 사용하는 방법**을 공부하고 실험하기 위한 공간입니다.

Spring, 실행 가능한 코드 자체가 목적이 아니라  
Claude Code의 다음 개념들을 이해하고 직접 실험하는 것이 목표입니다.

## 🎯 Study Focus

- claude.md (프로젝트 전역 규칙)
- skill (반복 지식 / 패턴 정의)
- command (사람이 사용하는 명령 인터페이스)
- hook (자동 개입 규칙)
- agent (역할 분리와 책임 분담)

## 🧪 How This Repo Is Used

- 실제로 실행되지 않아도 되는 **가벼운 프로젝트 구조**
- 코드보다 **문서와 규칙의 영향**을 관찰
- 커밋 단위 = 실험 단위
- “Claude가 어떻게 반응하는지”를 기록하고 비교
